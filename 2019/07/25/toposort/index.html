算法笔记_拓扑排序

发表于 2019-07-25 | 更新于 2019-07-26

拓扑排序是将一个有向无环图按一定方式排序的算法 配合递推/DP使用
同时也能够判断一个有向图中是否有环

1.引入

有些对于有向无环图的问题中 一个点的状态只有在所有能到达这个点的点的状态都被确定后 才能够根据这些点被确定
而拓扑排序就是对这些点进行排序 使得对于每一个点都排在所有能到达它的点之后

有点抽象 举个例子: 学习一个知识之前一定要会它的所有前置知识 有些知识不需要前置知识
拓扑排序可以给出学习这些知识的顺序

排序的结果成为拓扑序
同一张图可以有不同的拓扑序

2.实现

对每一个点计算它的入度(即有多少条边的到达这个点的)
首先 入度为0的点相当于没有”前置知识” 可以一开始就处理完这个点的状态(就像DP预处理)
我们先把这些点扔进一个FIFO的数据结构(q就是队列)里方便后面处理

之后 我们不断从队列里取出队首元素
由于该元素是已经处理完的点(状态已知) 我们可以视为这个点对 需要由该点的状态推得状态的点 约束已经去掉了 可以看做把这个点的所有出边删除 于是把所有出边到达的点的入度-1即可

相当于每学完一个知识a 把所有需要这个知识a才能学的知识 去掉了需要学a的这个前提 就是还剩下没学的前置知识少了一个

在上一步的过程中 如果出现了到达的点度数为0 那么这个点也就变成了状态已知的点 于是将其入队 来在以后推得之后的点

相当于把一个知识的所有前置知识都学完了 就可以学这个知识了

当队列为空时 如果还有点没被遍历到 那么就代表该有向图有环

举个例子配合理解:

5个点的入度分别为0 3 2 0 2
先将入度为0的1 4号点 入队&删边

5个点的入度分别为/ 1 0 / 2
(1 4号点已经出队了)
3号点入度为0了 入队&删边

5个点的入度分别为/ 0 / / 1
(1 3 4号点已经出队了)
2号点入度为0了 入队&删边

5入队

最后5什么都不做就出队了
结束

拓扑序14325

点出队的顺序就是拓扑序
