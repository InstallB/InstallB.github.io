codeforces游玩攻略

发表于 2019-08-28 | 更新于 2019-08-29

先把网址放上来:
www.codeforces.com

1.主界面

大概在图中解释了
Rating相当于是比赛积分 在后文会提到
contribution相当于社区贡献

这是最开始的界面



登陆后的界面是这样子的

在codeforces中 你可以在文章下发表评论 也可以评价别人的文章或评论(相当于顶/踩)
你的评论的顶/踩数会影响到你的contribution 注意言辞

2.比赛

codeforces上的比赛有两种赛制

另外codeforces也有一个比较特殊的功能 : hack

你可以通过查看一个人已经通过部分测试数据 (部分测试数据有时会故意设置的不强) 的代码 (双击排行榜/room中他的得分栏所对应题目的绿色数字 点击他的通过代码 就可以查看 , 点击hack it并上传测试数据或数据生成器即可hack) 尝试构造出一组数据 使得该程序返回错误的结果(或者超出时空限制等也可行)

另外 这里的题目没有部分分 你只有AC了这道题 才可以拿到分数

1.ACM赛制比赛
ACM赛制记录你的通过题目的总用时(解决这道题目的时间+unsuccessful submission罚时)和AC题目数量
比赛的排名按AC数量为第一关键字排序 AC的越多 排的越前面
AC数相同的人 总时间越少排在越前面
在ACM赛制的比赛结束后 会有12小时的open hack时间 在这段时间里 你可以去尝试hack所有人的代码
最后会将所有人的程序 对所有的原本给定的数据和用户的hack数据再测试一遍 称为system test

在这个过程中 你的原本AC了初始测试数据的程序可能会在某个测试点不通过 这样这道题目就会按你不通过来计算了

这时候你的程序还是能AC这道题目的话 才是真正的AC了这道题

每一次不AC的提交会使得这道题目获得10min的罚时
就是说如果你之后AC了这道题目 这个10min的罚时才会算在你的总时间里面
如果你到比赛结束都没有AC这道题 不通过的题目的罚时不会计算进总时间

Educational系列比赛和Div3均采用这种赛制 其他比赛都采用下文的CF赛制

2.Codeforces赛制
在codeforces赛制中 每道题目都会有一个初始给定分值 AC这道题目获得的分数会随时间等比例减少
AC的越早 获得的分数越高
此外 一道题目的测试数据分为Pretest和System Test
在比赛期间 只会评测Pretest部分是否通过 而Pretest有时会较弱 通过了Pretest不代表程序是正确的 只有通过了赛后评测的System Test你才能拿到这道题的分数(这个分数就是通过Pretest时的分数)
如果你的程序没有通过Pretest 那么在AC这道题目后这题的得分会被减去(50x不通过的提交总数)
如果你发现之间的程序通过了Pretest但是自己发现有问题 并重新交了一份代码 那么也会减去50分
被hack同理

如果你通过了Pretest并觉得自己的程序没有问题了 你可以选择lock这道题目 lock后你可以去hack别人 你可以查看所有和你在一个room里的所有参赛者的代码并尝试去hack他们
一次成功的hack会为你加上100分 一次失败的hack会减去50分
而lock题目之后就不能再交了 就是说即使你之前的程序发现了错误或者被hack 你也不能重新交一份代码上去 所以 hack有风险 锁题需谨慎啊

在比赛结束后 会对所有通过Pretest的程序测试 System Test+Hack数据 通过这些数据后 你才能拿到比赛期间显示的你这道题获得的分数 如果在这些测试数据挂了 这题就直接没分了 即Failed System Test(FST)

排名就是按照总分排名
最后按照总分排名

3.Rating
每一个人初始Rating为1500

根据每场比赛的排名和你的Rating 会计算出你的Rating的改变量 并改变你的Rating

Rating决定了你的名字颜色
Rating也决定了你能参加的比赛
单独Division2的比赛 : Rating<2100
Division3的比赛 : Rating<1600
一场比赛同时有Div1和Div2时
Div1: 1900及以上参加 / Div2 : 1900以下参加
Div1的前几题和Div2的后几题时相同的 Div1在后面加了几道难题 而Div2在前面加了几道简单题

高Rating还可以有一些特别的权力

<1200 : Newbie 灰
1200~1399 : Pupil 绿
1400~1599 : Specialist 青
1600~1899 : Expert 蓝
1900~2099 : Candidate Master 紫
2100~2299 : Master 橙
2300~2399 : International Master 橙
2400~2599 : Grandmaster 红
2600~2999 : International Grandmaster 红
3000及以上: Legendary Grandmaster 黑红

可以去装个RP在比赛期间预测rating : link
